## 定义
> 一种针有序区间内的O(logN)搜索方式，常见与已经排好序的Array

## 两大原则
1. 每次都要缩减搜索区域
2. 每次不能排除正确答案

## 三个模板(l = start;r = right)
+ 找一个准确值
  + 循环条件：`l <= r`
  + 缩减搜索空间：`l = mid + 1;` `r = mid - 1;`
+ 找一个模糊值(比4大的最小值)
  + 循环条件：`l < r`(若有=，会陷入死循环)
  + 缩减搜索空间： `l = mid,r = mid -1`或者`l = mid+1,r = mid`(根据原则判断)
  + mid的计算方法根据具体情况更改
+ 万用型(最接近2的值)
  + 循环条件：`l < r - 1`
  + 缩减搜索空间：`l = mid,r = mid`
